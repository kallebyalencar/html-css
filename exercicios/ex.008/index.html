Principais formatações:

Negrito/Destaque:

Ex: Meu nome é Johnny Kalleby

→ Neste caso, utilizou de uma forma não semântica... foi utilizado a tag <b> e </b>

→ Já, se procura uma forma semântica, usamos nesse caso: <strong> e </strong>
Ex: Minha namorada se chama: Tainá Santos

Nas duas foi usada a formatação-Negrito , destinguindo sua semântica.

Itálico:

Nesta frase, temos um termo em itálico (Não Semântico):
Ex: Está muito quente em juazeiro do Norte, não é verdade?
→ Utilizou a tag <i> e </i>

Nesta frase, temos um termo em ênfase usando a tag <em> e </em> (Sêmantica).

Texto Marcado:

Podemos criar também um texto marcado a tag <mark> e </mark>

E no outro parágrafo, temos outro texto marcado

Texto grande e pequeno:

Estamos criando um texto grande e um texto pequeno nesse parágrafo.
→ Tag "Big" foi obsoleta.
→ Letra pequena: tag "Small"

Texto Deletado:

Podemos marcar um texto como excluido para indicar que ele deve ser lido, porém não considerado.
Com a tag "DEL"

Texto Inserido:

Podemos marcar um texto como inserido para dar um ênfase e indicar que ele foi adicionado depois
Com a tag "INS"

Existe também o sublinhado com a tag "U" (não semântica)

Texto Sobrescrito:

Para inseriri coisas do tipo x2+3
Com a tag: "SUP"

Texto subscrito:

para inseriri coisas do tipo H2O
Com a tag "SUB"
